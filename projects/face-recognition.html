PCA + SVM 人脸识别系统

薛雅文2024
项目概况
项目类型线性代数课程拓展项目
技术栈MATLAB, PCA, SVM, MATLAB App Designer
核心功能27人人脸识别、实时摄像头采集、GUI应用
数据规模27人 × 6–8张/人，56×46像素灰度图

1   项目背景

本项目在MATLAB环境下构建了一套完整的PCA降维 + One-vs-All多分类SVM人脸识别系统，支持27人识别分类，并通过MATLAB App Designer开发了图形化应用界面，支持摄像头实时采集和马赛克特效。

2   算法流程

  1. 数据读取（ReadFace.m）：读取56×46灰度图，展平为2576维行向量，标注类别标签
  2. PCA降维（PCA.m）：中心化→协方差矩阵→特征分解→投影至k维子空间
  3. 特征归一化（scaling.m）：Min-Max缩放至[-1, 1]区间，消除量纲影响
  4. SVM训练（multiSVMtrain.m）：27个One-vs-All二分类SVM模型（fitcsvm）
  5. 预测推理（multiSVM.m）：Bagging后验概率投票选出类别
  6. GUI应用（xtw.mlapp）：MATLAB App Designer界面，集成拍照+预测

3   核心代码实现

3.1   数据读取与预处理

ReadFace.m – 批量读取人脸样本并展平
function [trainData, trainLabel, testData, testLabel] = ReadFace()
    n_person = 27;    % 27个人
    n_train = 6;      % 每人6张训练
    n_test = 2;       % 每人2张测试
    img_h = 56; img_w = 46;
    dim = img_h * img_w;  % 2576维

    trainData = zeros(n_person*n_train, dim);
    trainLabel = zeros(n_person*n_train, 1);

    for p = 1:n_person
        for s = 1:n_train
            filename = sprintf('照片库/z%d_%d.jpg', p, s);
            img = imread(filename);
            if size(img, 3) == 3
                img = rgb2gray(img);
            end
            img = imresize(img, [img_h, img_w]);
            trainData((p-1)*n_train+s, :) = ...
                double(reshape(img, 1, []));
            trainLabel((p-1)*n_train+s) = p;
        end
    end
    % 测试集同理...
end

3.2   PCA主成分分析

PCA.m – 协方差矩阵特征分解降维
function [pcaA, V, conVec] = PCA(A, k)
% 输入: A -- 样本矩阵(N x 2576), k -- 目标维度
% 输出: pcaA -- 降维后特征, V -- 主成分向量

[row, col] = size(A);

% Step 1: 去除均值（中心化）
meanA = mean(A);
meanVec = A - meanA;

% Step 2: 计算协方差矩阵 (使用 A'A 替代 AA' 加速)
conVec = meanVec' * meanVec / row;

% Step 3: 计算前k个最大特征值对应的特征向量
[V, D] = eigs(conVec, k);

% Step 4: 特征向量归一化为单位向量
for i = 1:k
    V(:,i) = V(:,i) / norm(V(:,i));
end

% Step 5: 将原始样本投影到k维子空间
pcaA = meanVec * V;

PCA的数学原理：设样本矩阵 X ∈ mathbbRN × d（d=2576），中心化后的协方差矩阵：

C = (1)/(N) barXT barX ∈ mathbbRd × d

取前k个最大特征值对应的特征向量 V = [v1, …, vk]，降维结果：

Y = barX · V ∈ mathbbRN × k

3.3   Min-Max特征归一化

scaling.m – Min-Max归一化至[-1,1]
function [scaledData, minVals, maxVals] = scaling(data)
    minVals = min(data);
    maxVals = max(data);
    range = maxVals - minVals;
    range(range == 0) = 1;  % 避免除零
    scaledData = 2 * (data - minVals) ./ range - 1;
end

3.4   One-vs-All多分类SVM训练

multiSVMtrain.m – 27个二分类SVM训练
function models = multiSVMtrain(X_train, y_train)
    y_labels = unique(y_train);
    n_class = length(y_labels);  % 27
    models = cell(n_class, 1);

    for i = 1:n_class
        % 构建二分类标签: 当前类=1, 其余=-1
        y_binary = -1 * ones(size(y_train));
        y_binary(y_train == y_labels(i)) = 1;

        % 训练SVM (线性核 + 后验概率)
        models{i} = fitcsvm(X_train, y_binary, ...
            'KernelFunction', 'linear', ...
            'FitPosterior', true);
        disp(['已训练: 类别 ', num2str(y_labels(i)), ...
              ' (', num2str(i), '/', num2str(n_class), ')']);
    end
end

3.5   Bagging概率投票预测

multiSVM.m – Bagging后验概率投票
function [pred, acc] = multiSVM(X_test, testlabel, y_train, models)
    y_labels = unique(y_train);
    n_class = 27;
    test_num = size(X_test, 1);
    y_predict = zeros(test_num, 1);

    for i = 1:test_num
        bagging = zeros(n_class, 1);
        for j = 1:n_class
            model = models{j};
            [~, rat] = predict(model, X_test(i,:));
            bagging(j) = bagging(j) + rat(2);
        end
        [~, maxp] = max(bagging);
        y_predict(i) = y_labels(maxp);
    end

    right = sum(y_predict == testlabel);
    acc = right / test_num;
    pred = y_predict;
    fprintf('识别准确率: %.2f%%\n', acc * 100);
end

3.6   单图预测接口

PredictImg.m – 新图像预测
function name = PredictImg(img_path, V, meanA, models, y_train)
    names = {'赵某某','肖某某','陈某某', ...
             '薛雅文', ...}; % 27人姓名列表

    img = imread(img_path);
    img = rgb2gray(img);
    img = imresize(img, [56, 46]);
    vec = double(reshape(img, 1, []));

    % PCA投影到训练时的子空间
    vec_centered = vec - meanA;
    feature = vec_centered * V;

    % 归一化 + SVM预测
    [pred, ~] = multiSVM(feature, 0, y_train, models);
    name = names{pred};
    fprintf('识别结果: %s\n', name);
end

5   迭代优化历程

版本迭代
  1. 基础PCA + 最近邻：仅PCA降维+欧氏距离KNN分类，准确率~70%
  2. 引入SVM：One-vs-All策略训练27个线性SVM，准确率显著提升
  3. Bagging投票：后验概率Bagging替代硬标签投票，提高鲁棒性
  4. GUI应用：MATLAB App Designer图形界面+摄像头实时采集+马赛克特效
