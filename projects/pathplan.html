激光雷达路径规划小车

薛雅文 2024

本项目以 Ackermann 转向结构小车为载体，搭载 N10 激光雷达（TOF 测距原理）实现室内环境感知。 设计了静态全局路径规划（多帧扫描建图 + 人工势场法）与动态目标检测（RANSAC 圆拟合）两种工作模式。 创新性地提出了距离—光强融合检测方法，解决黑色目标物距离数据不可靠的工程难题。

关键词：N10 激光雷达, TOF, RANSAC, 人工势场法, Ackermann 转向, 距离-光强融合

1.1   N10 激光雷达数据帧格式

N10 以固定帧格式通过 UART 输出数据。每帧包含帧头（0x54）、数据类型（0x2C）、 测量点数（12 点/帧）、起始角度、结束角度以及 12 组（距离 + 光强）数据， 末尾附 CRC 校验。每次旋转输出约 30 帧，合计约 360 个角度采样点。

2   静态路径规划

2.1   环境建图

小车在原地旋转采集多帧 LiDAR 数据，将极坐标 (r, θ) 转换为笛卡尔坐标 (x, y)， 拼合为二维散点地图。MATLAB 中对地图进行栅格化（分辨率 20 mm/格）， 标记为"空闲"（free）、"占据"（occupied）或"未知"（unknown）三类。

2.2   人工势场法

在栅格地图上构建人工势场：目标点产生引力场，障碍物产生斥力场。 合力方向即为期望运动方向。势场函数定义为：

Uatt(q) = ½ · ξ · ‖q − qgoal‖²
Urep(q) = ½ · η · (1/ρ(q) − 1/ρ₀)²    if ρ(q) ≤ ρ₀

从起点沿负梯度方向迭代搜索，步长自适应缩小，直至到达目标点。 为避免局部极小值，引入随机扰动与回溯机制。

2.3   路径跟踪

规划路径离散化为一系列路径点，MCU 按序跟踪。 Ackermann 转向模型下，根据当前位姿与目标路径点的偏差计算舵机转角 δ：

δ = atan(L · sin(α) / (ld/2 + L · cos(α)))

其中 L 为轴距，ld 为前视距离，α 为目标点相对航向角。

3   动态目标检测：RANSAC 圆拟合

3.1   算法原理

目标检测场景中，圆柱形目标物（直径已知）出现在雷达视野内。 传统最小二乘拟合对离群点敏感，故采用 RANSAC（Random Sample Consensus）：

  1. 从 LiDAR 点云中随机抽取 3 个点
  2. 由 3 点确定唯一圆（圆心 + 半径）
  3. 统计距拟合圆距离小于阈值 ε 的内点（inlier）数量
  4. 重复 N 次迭代（N 由置信概率与内点比例估算），选择内点最多的模型
  5. 用所有内点重新最小二乘拟合，得到精确的圆心坐标与半径

3.2   三点定圆公式

给定三点 (x₁,y₁), (x₂,y₂), (x₃,y₃)，圆心 (a,b) 与半径 r 的解析解：

a = [(x₁²+y₁²)(y₂−y₃) + (x₂²+y₂²)(y₃−y₁) + (x₃²+y₃²)(y₁−y₂)] / D
b = [(x₁²+y₁²)(x₃−x₂) + (x₂²+y₂²)(x₁−x₃) + (x₃²+y₃²)(x₂−x₁)] / D
D = 2[x₁(y₂−y₃) + x₂(y₃−y₁) + x₃(y₁−y₂)]

4   距离—光强融合检测

N10 雷达基于 TOF 原理测距，同时输出每个角度的光强（intensity）值。 在实际测试中发现：黑色目标物表面对 905 nm 激光反射率极低， 导致 TOF 测距结果不稳定（距离跳变或置零）。

关键创新：利用光强数据弥补距离数据的缺陷。 黑色表面虽距离不可靠，但其光强值显著低于白色背景—— 可用光强阈值初步判定"疑似黑色目标"所在角度区间， 再结合相邻角度的有效距离数据插值估算目标距离。 融合策略：

  1. 遍历一圈扫描数据，标记光强低于阈值 Ith 的角度段
  2. 对标记段两端的有效距离进行线性插值，得到该方向的估计距离
  3. 将估计坐标输入 RANSAC 圆拟合，验证是否符合目标直径

该方法在黑色圆柱目标上的检测成功率由单纯 TOF 距离模式的约 40% 提升至 85% 以上。

  • 搭建 Ackermann 转向小车 + N10 LiDAR 硬件平台

  • 实现 RANSAC 圆拟合动态目标检测（3 点定圆 + 迭代优化）
