仿生多模态智能台灯

薛雅文、郑皓文、严海轩、令狐子安 2025

本项目包含两个子课题。子课题一从控制理论出发，对 Buck 变换器在 PI 控制下的稳定性进行完整分析， 综合运用 Routh 判据、根轨迹法、Nyquist 稳定判据与 Bode 图四种方法，并在 PSIM 仿真与 STM32 数字控制中验证。 子课题二设计并实现一款仿生多模态智能台灯——以 Apple Elegant 为概念灵感， 搭载 Raspberry Pi CM5 计算模块与三自由度机械臂，融合计算机视觉（MediaPipe 手部追踪）、 语音识别与手势识别三种交互模式，实现手部跟随照明、桌面宠物与亮度调节等功能。

关键词：Buck 变换器, PI 控制, 稳定性分析, Raspberry Pi CM5, MediaPipe, 三自由度机械臂, 有限状态机

1   子课题一：Buck 变换器 PI 控制与稳定性分析

1.1   Buck 电路建模

Buck 变换器的小信号传递函数为二阶系统，以占空比 d̂ 为输入、输出电压 v̂o 为输出。 闭环 PI 控制器传递函数为 G_c(s) = K_p + K_i/s，与电力级传递函数级联后构成闭环。

1.2   四种稳定性分析方法

方法核心思路结论
Routh 判据 闭环特征方程系数列表，判断第一列符号变号次数 给出 Kp, Ki 的稳定域边界
根轨迹法 变化 Kp 时闭环极点在 s 平面的轨迹 极点穿越虚轴时的临界增益
Nyquist 判据 开环频率响应曲线对 (−1, j0) 的包围数 系统增益裕度与相位裕度
Bode 图 开环幅频与相频特性 穿越频率处增益裕度 ≥ 6 dB，相位裕度 ≥ 45°

四种方法一致给出稳定边界条件：负载电阻 R < 1.25 Ω 时闭环不稳定。

1.3   PSIM 仿真与 Python 验证

在 PSIM 中搭建 Buck 电路闭环模型，分别施加阶跃与斜坡负载扰动，验证 PI 控制的动态响应。 同时使用 Python（scipy.signal）绘制根轨迹、Bode 图与 Nyquist 图， 与理论推导结果完全吻合。

1.4   STM32 数字 PI 实现

在 STM32 上以数字 PI 控制器驱动 Buck 变换器。ADC 采样输出电压， 经差分方程 u[k] = u[k-1] + K_p(e[k]−e[k-1]) + K_i·e[k] 计算占空比， 通过 PWM 输出至 MOSFET 驱动。采样频率与 PWM 频率均为 100 kHz。

2   子课题二：仿生多模态智能台灯

2.1   整体方案

以 "Apple Elegant" 极简工业设计风格为灵感，台灯主体为三自由度串联机械臂 （底座旋转 + 大臂俯仰 + 灯头俯仰），末端集成 LED 照明模组与摄像头。 系统核心为 Raspberry Pi CM5（4-core Cortex-A76, 8 GB RAM）， 运行 Linux 操作系统，承担视觉推理、语音识别与状态机调度等计算密集任务。

2.2   系统架构

模块实现说明
计算核心RPi CM5 + IO Board运行 Python 主程序与 MediaPipe
舵机驱动PCA9685 (I²C)16 通道 PWM，驱动 3 个 SG90 舵机
视觉输入USB Camera30 fps，用于手部追踪与手势识别
语音输入USB 麦克风支持离线唤醒词与语音命令
照明高亮 LED 模组PWM 调光，舵机控制照射方向

2.3   有限状态机

软件采用有限状态机（FSM）架构管理三种模式间的切换，避免模式冲突：

  • 手部跟随模式：MediaPipe 实时检测手部关键点（21 landmarks）， 计算手掌中心在图像中的坐标，经反向运动学映射为三轴舵机角度， 灯头实时追踪手部移动。
  • 桌面宠物模式：语音指令驱动预设动作序列（点头、摇头、呼吸灯效等）， 赋予台灯拟人化交互体验。
  • 亮度调节模式：通过手势（张开/握拳/旋转）控制 LED 亮度等级， 手势特征由 MediaPipe 手部关键点的几何关系提取。

2.4   MediaPipe 手部追踪

Google MediaPipe Hands 模块输出 21 个手部关键点的三维坐标（归一化到 [0,1]）。 取手掌中心（landmark 0, 9 的中点）作为目标位置，经 PID 平滑后映射到舵机角度空间。 实测在 RPi CM5 上帧率约 25 fps，延迟 < 80 ms，满足实时跟随需求。

2.5   逆运动学

三自由度串联臂的逆运动学可解析求解。给定目标点 (x, y, z)——由相机坐标系经标定矩阵 映射至臂基座坐标系——依次计算底座旋转角 θ1 = atan2(y, x)， 大臂俯仰角 θ2 与灯头俯仰角 θ3 由余弦定理联立求解。
