工程设计课程项目——Gamma型斯特林发动机联合仿真

薛雅文2025
项目概况
课程工程设计（热力学方向）
研究课题Gamma型斯特林发动机的ADAMS-MATLAB联合仿真分析
核心工具SolidWorks / ADAMS / MATLAB Simulink
设计目标输出功率≥0.5 W，额定转速1500 r/min

本项目以Gamma型斯特林发动机为研究对象，从热力学理论出发，经历结构参数设计→三维建模→多体动力学仿真→联合仿真平台搭建的完整工程流程，实现了热力学计算与机械动力学的跨域耦合仿真。

1   斯特林循环热力学基础

斯特林循环由四个基本过程组成，是一种外燃式闭式循环：

  1. 等温压缩（1→2）：气体在低温侧被等温压缩，对外放热 \[ Q12 = W12 = -mRgTC · ln(Vmax)/(Vmin) = -0.4319 J \]
  2. 等容吸热（2→3）：气体通过回热器从低温侧流向高温侧，等容吸热 \[ Q23 = mcV(TE - TC) = 0.5968 J, W23 = 0 \]
  3. 等温膨胀（3→4）：气体在高温侧等温膨胀对外做功 \[ Q34 = W34 = mRgTE · ln(Vmax)/(Vmin) = 0.7199 J \]
  4. 等容放热（4→1）：气体通过回热器返回低温侧，等容放热 \[ Q41 = -mcV(TE - TC) = -0.5968 J, W41 = 0 \]

单次循环理论做功：

\[ Wi = W34 + W12 = mRg(TE - TC) · ln(Vmax)/(Vmin) = 0.2880 J \]

在回热器效率etaR = 1的理想条件下，热效率达到卡诺效率：

\[ eta = (Wi)/(Qin) = (TE - TC)/(TE) = 40.0% \]

2   史密特理论建模

项目采用史密特（Schmidt）理论对Gamma型发动机进行准稳态热力学分析。核心假设包括：等温压缩/膨胀、理想气体、完全再生热交换、正弦容积变化。

2.1   容积与压力的解析表达

以排气器活塞上止点为曲柄角θ = 0^°，膨胀空间与压缩空间瞬时容积为：

\[ VE = (VSE)/(2)(1 - cosθ), VC = (VSE)/(2)(1 + cosθ) + (VSC)/(2)[1 - cos(θ - α)] \]

瞬时压力由压力方程给出：

\[ p = (pmean√1 - δ2)/(1 - δcos(θ - φ)) \]

其中δ = B/S，B、S为由温度比tau = TC/TE、行程容积比kappa和死容积比chi确定的参数。

5   SolidWorks三维建模与ADAMS仿真

5.1   三维结构模型

在SolidWorks中精确构建Gamma型斯特林发动机的连杆、飞轮、活塞等关键部件，将Parasolid格式导入ADAMS进行动力学仿真。

5.2   ADAMS运动副与约束设置

在ADAMS中定义多种运动副实现真实的机械运动关系：

  • 大地固定副：提供全局参考系，避免冗余自由度导致仿真不收敛
  • 转动副：连杆/飞轮与地面之间的旋转运动关系
  • 平移副：活塞与气缸之间的往复运动关系
  • 圆柱副：连杆与活塞销连接处的复合运动

定义4个状态变量：torque1（输入力矩，从Simulink获取）、angle1（轴转角）、velocity1（连杆转速）、force1（输入力），实现ADAMS与MATLAB的数据交互。

6   ADAMS-MATLAB联合仿真

6.1   Simulink热力模型

在MATLAB/Simulink中编写plant.m S-Function，实现史密特理论的实时计算：转角θ作为输入，根据史密特公式计算瞬时气缸压力p、活塞作用力Fp = (p - pair)Ap、曲柄转矩Tq = Fp r(sinθ - cosθ)，输出转矩反馈至ADAMS驱动机械系统运动。

6.2   仿真结果与分析

转矩随时间变化曲线
转速随时间变化曲线

初始仿真转速约90 r/min，低于设计目标。分析原因：

  • t < 0.1 s时初始力矩偏小（5 N + 0.3 N·m），启动能量不足
  • 进入稳态后持续计算输出转矩偏小
  • 飞轮转动惯量设定影响转速稳定性

6.3   参数敏感性分析与优化

通过调整关键参数实现目标转速：

  • 增大活塞直径DP：增加有效受压面积，提高单次循环做功
  • 增大气缸长度：增大行程容积VSE，提升循环功率
  • 优化飞轮质量与转动惯量：增大飞轮惯量可储存更多动能，稳定转速

经过多轮参数调整，最终实现1500 r/min的设计目标转速。

项目成果
  • 建立了Gamma型斯特林发动机的完整热力学理论模型（史密特理论）
  • 通过SolidWorks建模→ADAMS多体动力学仿真→MATLAB/Simulink热力耦合，实现跨域联合仿真
  • p-V闭合环路面积定量表征单次循环做功，理论热效率40%（卡诺极限）
  • 参数敏感性分析明确了活塞直径、气缸长度、飞轮惯量对转速的影响机制
  • 最终实现设计目标功率≥0.5 W、额定转速1500 r/min
